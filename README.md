## Описание страниц

### Главная (index.html)
- Визитная карточка студента с фото и краткой информацией.
- Секция "Обо мне".
- Секция "Мои навыки" с прогресс-барами.
- Секция "Опыт/Проекты" с кликабельными карточками, которые ведут на репозитории GitHub.

### Проекты (projects.html)
- Сетка проектов с фильтрацией по типу (HTML, JS, React).
- При клике открывается модальное окно с описанием проекта, скриншотами и ссылкой на GitHub.
- Используется .projects-grid и .project-card с hover-эффектами.

### Учебный дневник (diary.html)
- Секция с прогрессом по задачам в виде карточек.
- Цветовое оформление: завершённые задачи — голубой, в процессе — жёлтый.
- Секция текущих курсов с прогресс-барами.
- Совпадает по стилю с главной страницей (skills-grid, progress-bar).

### Контакты (contacts.html)
- Форма для отправки сообщений с полями: Имя, Email, Сообщение.
- Валидация через required.
- Контактная информация: email, GitHub, местоположение.
- Кнопка отправки показывает alert и сбрасывает форму.
